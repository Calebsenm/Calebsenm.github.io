Configuracion de latex para trabajar en Llinux con vim o nano

De seguro ya tienes instalado el copilador de latex En esta o Ocacion Voy a configurar Para usarlo en Linux y con el editor latex
Ahora debes instalar
satura "Previsualizador de pdf"
Para sistemas Debian o Ubuntu y dervados
sudo apt install zathura
ahora copilador para latex

sudo apt install latexmk
ahora par copilar el archvo pdf puedes usar

latexmk - pdf

ahora para poder abrirlo puedes usar

zathura documento.pdf


Ademas si colocas
zathura -pdf -pvc documento.pdf

Ahora lo puedes tener como corriendo en segundo plano mientras editas el archivo y los cambios se van a guardar y mostrar en el pdf
