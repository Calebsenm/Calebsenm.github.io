Instalacion de Qtile

Para instalar qtile puedes intentarlo primer con pip3 pero debes instalar antes algunas dependencias :

sudo apt install xserver-xorg xinit
sudo apt install libpangocairo-1.0-0
sudo apt install python3-pip3 python3-xcffib python3-cairocffi

pip3 install qtile
o intenta

sudo pip3 install qtile

Ya deberias Tener Qtile si no funciona puedes intentarlo desde el codigo fuente de la siguiente manera:

git clone https://github.com/qtile/qtile.git
cd qtile
pip3 install .

Ves a siguiente directorio :

cd /usr/share/xsessions
crea el siguiente archivo que permite mostrar qtile en el inicio
touch qtile.desktop
ahora con un editor como nvim o nano edita el archivo tienes que privilegios

sudo vim qtile.desktop
ahora coloca el siguiente contenido en el archivo qtile.desktop

[Desktop Entry]
Name=Qtile
Comment=Qtile Session
Exec=qtile start
Type=Application
Keywords=wm;tiling

por ultimo debes configurar qtile en en path de la suguiente manera:
con nvim o nano

abre .bashrc

en la ultima linea vas a pegar :

export PATH=$PATH:/home/(aquiva Tu usuario)/.local/bin


Ya tienes lista la instalacion ya puedes cerrar seccion y iniciar en qtile
