Plan de estudio para aprender Go y luego enfocarse en el desarrollo web backend y en microservicios:

Tema Dificultad Ejercicio Descripción
Values Fácil 1 Crea una función que acepte dos valores enteros y retorne su suma, resta, multiplicación y división.
Values Intermedia 2 Crea una estructura que represente una persona y utiliza valores para inicializar una instancia de la estructura.
Values Difícil 3 Crea una función que acepte un valor entero y retorne si es par o impar.
Values Extrema dificultad 4 Crea una función que acepte una cadena y retorne si es un palíndromo o no.
Values Imposible 5 Crea una función que acepte dos valores de punto flotante y retorne si son iguales o no con un margen de error del 0.001.
