Proyectos de programación

Concepto Proyectos
Variables
  • Crea un programa que calcule el área de un rectángulo utilizando la fórmula "largo x ancho". El usuario debería poder ingresar los valores de largo y ancho.
  • Crea un programa que convierta una temperatura de grados Celsius a Fahrenheit o viceversa. El usuario debería poder elegir qué conversión realizar y ingresar el valor de la temperatura.
Tipos de datos
  • Crea un programa que almacene una lista de estudiantes y sus calificaciones en variables de diferentes tipos de datos (por ejemplo, cadena para el nombre del estudiante, entero para la calificación). El programa debería permitir al usuario agregar nuevos estudiantes a la lista y mostrar la lista con los nombres y calificaciones de los estudiantes.
  • Crea un programa que lea un archivo CSV que contenga datos sobre transacciones de ventas y almacene los datos en variables de diferentes tipos de datos (por ejemplo, cadena para el nombre del cliente, flotante para el monto de la transacción). El programa debería calcular las ventas totales para cada cliente y mostrar los resultados en una tabla.
Operadores
  • Crea un programa que calcule el perímetro de un rectángulo dado el largo y el ancho. El usuario debería poder ingresar los valores de largo y ancho.
  • Crea un programa que calcule el volumen de un cubo dado el largo del lado. El usuario debería poder ingresar el valor del largo del lado.
Control de flujo
  • Crea un programa que le pida al usuario que ingre
  • Crea un programa que le pida al usuario que ingrese su edad y muestre un mensaje indicándole si es o no mayor de edad para votar.
  • Crea un programa que le pida al usuario que ingrese un número y muestre todos los números primos entre 1 y el número ingresado.
